# TFG-Aplicación-interactiva-para-simulación-de-enfermedades-contagiosas-en-entornos-académicos
Este repositorio contiene la aplicación desarrollada como Trabajo de Fin de Grado por parte del estudiante Cristian Rey Márquez. La aplicación tiene el objetivo de simular de manera científica la propagación de un virus en escenarios personalizables creados por los usuarios.
